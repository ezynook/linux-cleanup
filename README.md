# Linux-Cleanup

## Support OS
* Debian
* RHEL
<details>
  <summary>
  What does script do?
  </summary>

</br>

* Clear PM2 logs
* Clear package caches
* inactive memory and swap
* logrotate 7 days
* Trim logs file
* Remove old kernels
* Remove the Composer cache
* Remove Node.js caches
* Remove Mock caches
* Tail Log Last Proccess

</details>


---
## Setup
```bash
curl -fsSL https://raw.githubusercontent.com/ezynook/linux-cleanup/main/install.sh | bash
```
## How to Use
```bash
$ linux-cleanup
```
